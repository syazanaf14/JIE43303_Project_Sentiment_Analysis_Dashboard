# JIE43303_Project_Sentiment_Analysis_Dashboard

Topic: Bilingual Sentiment & Review Integrity Dashboard

Overview:
An advanced NLP application designed to analyze sentiments and emotions in **English and Malay**. The system features a unique "Integrity Check" to verify if user ratings (1-5 stars) align with the written content.

Features:
- Bilingual Processing: Supports Malay and English text.
- Sentence-Level Analysis: Handles mixed feelings by breaking down paragraphs.
- Emotion Detection: Identifies Joy, Sadness, Anger, etc.
- Integrity Status: Detects mismatches between stars and sentiment.

Application:
https://jie43303-projectsentimentanalysis.streamlit.app/ 

Dataset:
Currently using Dataset from Kaggle, Shopee Customer Reviews + Ratings
